partAccordionMenu

  • Often used in combination with slidebars

Parent is always natural

  • Parent means it has childs
  • Natural means that page has content. A not natural page does not exist.
  • The class 'active' on a 'li' element is applied when the parent is showing childs
  • The class 'selected' on a 'li' element is applied when a parent with a natural is selected
  • Childs will only show when clicked the dropdown arrow
  • Besides the class 'partAccordionMenu', you can add more classes.

Accordion with content

  • Childs show when click on the parent or the arrow.
  • Class 'accordionMenuDropdown' can be applied with a span, h1, h2 or h3 element
  • Most commmon to use when you have a navigation with mutiple levels and the parrent doesn't have content in most cases.
  • The class 'active' on a 'li' element is applied when the parent is showing childs
  • Besides the class 'partAccordionMenu', you can add more classes.
