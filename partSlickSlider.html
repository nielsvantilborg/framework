partSlickSlider

Notes

  • You always need the class partSlickSlider
  • If you use multiple slick slickers, apply an extra class. For example 'partSlickSliderHeader', where you can apply CSS on and diviate on JS code.
  • Within the clean 'div' for every slide you can add HTML code to your desire. The image and div with class 'sliderCaption' are just examples what you can do.

Source

http://kenwheeler.github.io/slick/
