Parallax

Parallax top

Notes

  • Only apply this parallax to a fixed top margin.
  • background-position top is based on the height of the fixed element. No fixed element? Set it to 0.
  • Padding-top is based on the image ratio. For example: The image is 1440x400. 400/1440*100 = 27,777.. (always round down 27.7)

Demo

  • See top image :)

HTML

<div class="parallaxContainer parallaxContainerTop" style="background-image: url('https://placeimg.com/1440/400/nature');"></div>

CSS

.parallaxContainer {background-repeat: no-repeat; position: relative; background-attachment: fixed;} .parallaxContainerTop {background-position: center 90px; background-size: 100% auto; padding-top: 27.7%;} @media (max-width: 666px){ .parallaxContainerTop {padding-top: 0px; height: 200px; background-position: center center; background-attachment: scroll; background-size: cover;} }

Parallax middle

Notes

  • Only apply this parallax when the effect is in the middle of the page.
  • Padding-top can be set to whatever you want as height. We use padding-top in precentages to have a dynamic height. If the height is static or depends on the content. Just set padding-top to 0.
  • Tip: background-size: cover, uses the viewport, not the parrent to determine how the image will be applied.

Demo

HTML

<div class="parallaxContainer parallaxContainerMiddle" style="background-image: url('https://placeimg.com/1440/1000/nature');"></div>

CSS

.parallaxContainer {background-repeat: no-repeat; position: relative; background-attachment: fixed;} .parallaxContainerMiddle {background-position: center center; background-size: cover; padding-top: 34.72%;} @media (max-width: 666px){ .parallaxContainerMiddle {padding-top: 0px; height:200px;} }
