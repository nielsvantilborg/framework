partBreadcrumb

Notes

  • In this example the extra class 'partBreadcrumbTextLabel' and 'partBreadcrumbHomeIcon' on the 'ul' element are not needed if you have one unique breadcrumb.
  • Always remember if you use 'display:inline-block' there is a weird margin (left) bug on the li. So 5px is in this case 10px. The degree of deviation is dependent on the font size.

Breadcrumb basic

  • Home
  • Navigation name
  • Sub navigation name
  • Sub sub navigation name
  • Sub sub sub navigation name

HTML

<ul class='partBreadcrumb'> <li> <a hreflang='nl' title='Home' href='/'>Home</a> </li> <li> <a hreflang='nl' title='Koekenpannen' href='/koekenpannen/'>Navigation name</a> </li> <li> <a hreflang='nl' title='Koekenpannen' href='/koekenpannen/'>Sub navigation name</a> </li> <li> <a hreflang='nl' title='Koekenpannen' href='/koekenpannen/'>Sub sub navigation name</a> </li> <li class='selected'> <a hreflang='nl' title='Koekenpannen' href='/koekenpannen/'>Sub sub sub navigation name</a> </li> </ul>

CSS

.partBreadcrumb {text-align: left; margin-bottom: 40px;} .partBreadcrumb li {margin:5px 5px 5px 0px; display: inline-block;} .partBreadcrumb li:last-child {margin-right: 0px;} .partBreadcrumb li:after {content:'>'; margin-left: 5px;} .partBreadcrumb li:last-child:after {content:none;} .partBreadcrumb li a {} .partBreadcrumb > li > a {color: #666; text-decoration:none;} .partBreadcrumb > li.selected > a {color: #000; text-decoration:none;} .partBreadcrumb > li > a:hover {color: #000; text-decoration:underline;}

Breadcrumb with text label

  • Home

  • Sub navigation name

<ul class='partBreadcrumb partBreadcrumbTextLabel'> <li>You are here:</li> <li> <a hreflang='nl' title='Home' href='/'>Home</a> </li> <li> <a hreflang='nl' title='Koekenpannen' href='/koekenpannen/'>Navigation name</a> </li> <li class='selected'> <a hreflang='nl' title='Koekenpannen' href='/koekenpannen/'>Sub navigation name</a> </li> </ul>

CSS

.partBreadcrumb {text-align: left; margin-bottom: 40px;} .partBreadcrumb li {margin:5px 5px 5px 0px; display: inline-block;} .partBreadcrumb li:last-child {margin-right: 0px;} .partBreadcrumb li:after {content:'>'; margin-left: 5px;} .partBreadcrumb li:last-child:after {content:none;} .partBreadcrumb li a {} .partBreadcrumb > li > a {color: #666; text-decoration:none;} .partBreadcrumb > li.selected > a {color: #000; text-decoration:none;} .partBreadcrumb > li > a:hover {color: #000; text-decoration:underline;} .partBreadcrumbTextLabel li:first-child:after {content:none;}
